# 기능 목록

- [x] 게임 시작 메시지를 출력
- [x] 1~9 범위의 중복되지 않는 임의의 세자리 수를 생성
- [x] 세자리 숫자를 입력
  - [x] 입력받은 숫자에 대한 예외처리
- [] 입력받은 값에 대한 결과를 출력
  - [x] 두 배열이 모두 포함한 값의 개수를 알 수 있다
  - [x] 두 배열의 인덱스와 값이 일치하는 개수를 알 수 있다 (스트라이크)
  - [x] 볼의 개수를 알 수 있다
- [x] 게임 승리 메시지를 출력
- [x] 게임 재시작 커맨드를 입력
  - [] 재시작 커맨드에 대한 예외처리
- [] 게임을 재시작 할 수 있다
- [] 게임 종료
